Equipement matériel et configuration matériel

Globalement : rien à faire. Juste brancher l'ordinateur hp

Initialisation et configuration du système d'exploitation

Là encore trés peu de chose à faire. Il s'agit d'un système d'exploitation windows qui va se configurer tout seul et qui demande des inputs de temps à autre. Le seul point indispensable est de lui fournir une connexion réseau filaire ou WiFi. A la fin on dispose d'un ordinateur avec un compte administrateur. Une remarque. Si comme moi on utilise plusieurs ordinateurs, il est judicieux d'avoir un compte Microsoft et donc d'avoir ses fichiers et son environnement stocké sur OneDrive et non en local sur le l'ordinateur.

git status nous permet de voir que 1 fichier a été modifié (README.md) et qu'un fichier n'est pas encore suivi dans git. Notre action est de rajouter (add) toutes ces modifications dans le suivi git local et de vérifier le statut.

C:\Users\gfore\OneDrive\Documents\...\bootstrap\premiersite>git status

Après avoir ajouté les différents fichiers dans le suivi git local, on effectue un commit avec un message. Il se peut que la première fois nous rencontrions une erreur :

Pour résoudre ce problème d'identité il suffit d'effectuer les deux commandes recommandées par git :

C:\Users\gfore\OneDrive\Documents\...\bootstrap\premiersite>git config --global user.email "gforestier2000@yahoo.fr"

C:\Users\gfore\OneDrive\Documents\...\bootstrap\premiersite>git config --global user.name "gforestier2000"
